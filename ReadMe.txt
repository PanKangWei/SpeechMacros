



SpeechMacros
Windows 语音识别 宏

=====================
1. 程序说明
=====================

    SpeechMacros 包括一些语音识别的宏操作

=====================
2. 使用环境
=====================

     * 需要 Windows vista 、Windows 7 语音识别功能
     * 需要 WSRMacros.msi 下载链接：http://www.microsoft.com/en-us/download/details.aspx?id=13045
	虽然注明是适用于 vista 系统，经测试在 windows 7下可以正常安装和使用，Windows 8 未测试
